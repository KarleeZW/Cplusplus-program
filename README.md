# C++ Height & HeightRange Classes

This repository contains C++ classes that model height values and ranges of heights.  
It includes full unit conversion support and logic for comparing and validating height inputs.

## Files Included
- **height.h** – Class definition for `Height`
- **height.cc** – Implementation of height unit validation, conversion, getters, and setters
- **heightrange.cc** – Implementation of the `HeightRange` class for comparing two heights

## Features
- Validates height units: inches, feet, meters, centimeters
- Converts between all supported units
- Calculates tallest, shortest, and range width
- Checks whether a height falls inside a given range
